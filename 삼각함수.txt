삼각법 : 직각삼각형을 다루는 것

피타고라스 
a^2 + b^2 = c^2

삼각비 : 직각삼각형 변들의 비율
1:1:√2   3:4:5

삼각함수 : input 값으로 세타를 받고 output 값으로 삼각비를 갖는다

sin(θ) = b/c
cos(θ) = a/c
tan(θ) = b/a

cos -> a = c * cos(θ)
sin -> b = c * sin(θ)

호도법 : 호의 길이와 반지름의 길이가 같다면 1rad

360도 -> 2ㅠ rad
180도 -> ㅠ rad

1도 는 몇 라디안 ? ㅠ / 180 rad
70도 는 몇 라디안? 70 * ㅠ / 180
